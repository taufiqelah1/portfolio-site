## Getting Started

### Prerequisites

- Node.js (v14 or higher) - [Download](https://nodejs.org/)
- npm (comes with Node.js)

### Installation

1. Clone the repository or download the files
2. Install dependencies:
```bash
npm install
```

### Running the Project

Start the development server:
```bash
npm start
```

This will:
- Start a local server on `http://localhost:3000`
- Automatically open the site in your default browser
- Watch for file changes (with hot reload if using `npm run dev`)

### Alternative: Simple Server

If you don't have Node.js installed, you can also:

**Using Python:**
```bash
# Python 3
python -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000
```

**Or simply open `index.html` directly in your browser** (though some features may be limited without a server)

## Customization

### Personal Information
- Update the name, title, and description in the hero section
- Add your own image or keep the placeholder
- Update the about section with your story
- Modify the skills section with your technologies
- Add your actual projects with links
- Update social media links in the contact section

### Colors
The color scheme can be customized in `styles.css` using CSS variables:
```css
:root {
    --primary-color: #64ffda;
    --secondary-color: #ccd6f6;
    --text-color: #8892b0;
    --background-color: #0a192f;
    /* ... more colors */
}
```

### Content
- Replace placeholder text with your own content
- Update project links (GitHub, live demo)
- Add your email address in the contact section
- Update social media links
